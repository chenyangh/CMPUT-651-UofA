Download IMDB data from [kaggle](https://www.kaggle.com/iarunava/imdb-movie-reviews-dataset)
, unzip it and it will have the following structure. 
Assume ``aclImdb`` is under the root directory of this project.

```
aclImdb
    |-- train
        |-- pos
        |-- neg
        |-- unsup
    |-- test
        |-- pos
        |-- neg
```
